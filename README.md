# Spring Boot + SSE

## Requirements
- Docker 20.10.6
- Docker Compose 1.29.1
- Modern browser (tested in Mozilla 88.0 and Chrome 90.0.4430.93)

## Run demo
- Start docker containers (command: `docker-compose up -d`)
- Open site by url: http://localhost:8080
